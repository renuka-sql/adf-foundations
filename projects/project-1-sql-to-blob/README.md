# Project 1 – SQL to Blob Pipeline

## Overview

This project demonstrates a basic data movement pipeline using Azure Data Factory.

The pipeline exports data from Azure SQL to Blob Storage as a CSV file and loads it back into another SQL table.

## Architecture

Azure SQL → Copy Activity → Blob Storage  
Blob Storage → Copy Activity → Azure SQL
